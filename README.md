## Introduction

This repository contains the documentation for the [WepSIM project](https://wepsim.github.io).

The documentation uses [Honkit](https://honkit.netlify.app/) and [NPM](https://www.npmjs.com/):
* To build it, please use:
  ```bash
  npm install
  npm run build
  ```
* To deploy locally, please use:
  ```bash
  npm run serve
  ```
