# WepSIM documentation
This repository contains documentation for the [WepSIM project](https://wepsim.github.io).

The webpage uses:
* [Honkit](https://honkit.netlify.app/).
* [NPM](https://www.npmjs.com/)


To build the webpage please use:
```bash
npm install
npm run build
```

To deploy locally please use:
```bash
npm run serve
```
